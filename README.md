# Syntora – Student Management Portal

Syntora is a role-based web application designed to streamline academic record management within an institution. The system enables administrators, faculty, and students to securely access and manage student-related information through structured dashboards and controlled access.

## Features
## Role-Based Authentication

Secure login system for Admin, Faculty, and Students

Access control based on user roles

Dashboard-specific visibility of data

## Admin Module

Manage student records

Add/update academic information

Monitor attendance and performance

Publish notifications

## Faculty Module

Upload and manage attendance

View student performance data

Update project or academic records

## Student Module

View personal academic records

Track attendance and performance

Access notifications and updates

## Tech Stack

Backend: Java (JSP, Servlets)

Frontend: HTML, CSS, JavaScript

Database: MySQL

Server: Apache Tomcat

## Key Implementation Highlights

Implemented session-based authentication for secure user access

Designed database schema for structured student data management

Integrated JSP-Servlet architecture for backend request handling

Deployed using Apache Tomcat server

## Objective

The goal of Syntora is to provide a centralized, efficient, and secure platform for managing student academic workflows while reducing manual administrative effort.
